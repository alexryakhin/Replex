# Images Directory

## Directory Structure

- `screenshots/` - Place your iPhone app screenshots here
  - Recommended naming: `screenshot-1.jpg`, `screenshot-2.jpg`, `screenshot-3.jpg`
  - Or use descriptive names: `workout-tracking.jpg`, `exercise-library.jpg`, `progress-stats.jpg`

- `icons/` - Place your SVG icons here
  - Recommended naming: `icon-name.svg` (e.g., `logo.svg`, `feature-workout.svg`)

## Image Guidelines

### Screenshots
- Format: JPG or PNG
- Aspect Ratio: 9:19.5 (iPhone 14 Pro standard)
- Recommended size: 300-400px width (will scale responsively)
- Optimize images for web (use tools like ImageOptim or Squoosh)

### Icons
- Format: SVG (preferred) or PNG
- Keep SVG files optimized
- Use consistent sizing (typically 24x24, 32x32, or 48x48 viewBox)
